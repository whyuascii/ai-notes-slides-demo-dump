# AI Agents, Claude Code, and Practical Workflows

A comprehensive Slidev presentation about using AI agents effectively in modern development workflows.

## Getting Started

### Prerequisites

-   pnpm (or npm/yarn)

### Installation

```bash
pnpm install
```

### Running the Presentation

Start the development server:

```bash
pnpm dev
```

The slides will automatically open in your browser at <http://localhost:3030>

### Building for Production

To build the slides for deployment:

```bash
pnpm build
```

The built files will be in the `dist` directory.

### Exporting

To export slides as PDF:

```bash
pnpm export
```
